# Kullanım Durumları (Use Cases)

LlamaIndex, çok çeşitli yapay zeka uygulamaları için güçlü yetenekler sunar. İhtiyaçlarınıza özel olarak LlamaIndex'ten nasıl yararlanabileceğinizi öğrenmek için aşağıdaki kullanım durumlarını keşfedin:

-   [İstem Verme (Prompting)](/python/framework/use_cases/prompting) - LlamaIndex ile gelişmiş istem tekniklerini öğrenin
-   [Soru-Cevap (RAG)](/python/framework/use_cases/q_and_a) - Veri getirme ile zenginleştirilmiş oluşturma (retrieval-augmented generation) sistemleri oluşturun
-   [Sohbet Robotları (Chatbots)](/python/framework/use_cases/chatbots) - Diyalog bazlı yapay zeka uygulamaları geliştirin
-   [Yapılandırılmış Veri Çıkarma](/python/framework/use_cases/extraction) - Yapılandırılmamış metinlerden yapılandırılmış bilgiler çıkarın
-   [Ajanlar (Agents)](/python/framework/use_cases/agents) - Otonom yapay zeka ajanları geliştirin
-   [Çok Modlu (Multi-Modal) Uygulamalar](/python/framework/use_cases/multimodal) - Metin, resim ve diğer veri türleriyle çalışın
-   [İnce Ayar (Fine-Tuning)](/python/framework/use_cases/fine_tuning) - Modelleri kendi spesifik kullanım durumlarınız için özelleştirin
